# zm-tree-org

## 介绍
一个简易版组织架构图，组件依赖于vue-org-tree， 在此基础上将部分源代码进行优化修改。可鼠标拖动拖拽，以及鼠标滚轮缩放，并且支持拖动节点改变树结构。
- 架构图支持拖拽和通过鼠标滚轮缩放
- 支持新增/删除节点
- 支持编辑节点名称
- 支持拖动节点改变树结构
- 支持自定义右键菜单
- 支持slot自定义节点渲染内容
- 支持slot自定义展开按钮渲染内容

### 安装
```
npm install zm-tree-org --save
# or 
yarn add zm-tree-org
```

### 引入
```
import Vue from 'vue';
import ZmTreeOrg from 'zm-tree-org';
import "zm-tree-org/lib/zm-tree-org.css";

Vue.use(ZmTreeOrg);
```
